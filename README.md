# Markdown 图片素材自动化整理工具

## 🚀 简介
这是一个轻量级的自动化脚本，专门用于提取 Markdown 文档中的网络图片源，自动下载到同名的本地子目录中，并将文档内的外链替换为本地相对路径，实现文档图片的彻底本地化存储。

## 📂 目录结构
- **`create.py`**: 核心 Python 脚本代码。
- **`一键处理图片.bat`**: 🌟 **双击执行入口**，免除任何行命令操作。
- **`Clippings/`**: **（放置区）** 放入所有你需要被处理的 `.md` 草稿或截取文章。
- **`assets/`**: **（产出区）** 脚本会自动在这个目录下生成与文章完全同名的专属文件夹，用于存放下好的图片。

## 💡 能干什么？
1. **自动归档**：根据 `Clippings` 里的文件名，在 `assets` 下对应生成同名专属图片库。
2. **离线化下载**：扫描全篇，把以 `http` 开头的网络配图无缝下载到本地。
3. **一键换链**：自动修改原 md 文稿，把旧的网络图床长链接切换为优雅的本地路径。
4. **强迫症清理**：如果你不要某篇 `.md` 源文件并把它删了，再次运行脚本时，它会自动帮你把 `assets` 里遗留的失效配图文件夹一并无痕销毁。

## 🎯 怎么用？
1. 把准备好的 `.md` 文件塞进 **`Clippings`** 文件夹。
2. 回到外层，直接双击 **`一键处理图片.bat`**。
3. 等待命令行提示运行完毕，按任意键关闭即可。打开文档检查成果！
