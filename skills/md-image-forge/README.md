# Markdown 图片素材自动化整理工具 (md-image-forge)

## 简介
这是一个针对 **Markdown 资源的完全托管式整理工具**：
- 自动提取文档中的网络图片并下载。
- **接管杂乱的本地导出图片**，将引用的素材迁移至结构化的资源库中。
- 将文档内的图片链接无缝替换为标准的相对路径（提供 URL 编码保护）。
- **全自动清理**无用图片、冗余副本及空文件夹，保持工作区极致整洁。

## 目录结构
- **`scripts/create.py`**: 核心 Python 处理逻辑。
- **`SKILL.md`**: AI 代理技能定义，支持自动化任务调度。
- **`process_images.bat`**: 手动执行入口，双击即用。
- **项目根目录下 /Clippings/**: **（共享输入区）** 放入你的 `.md` 文档及任何关联的图片文件夹。
- **项目根目录下 /Clippings/assets/**: **（托管资源区）** 脚本会自动在此生成与文章同名的子目录存放图片。

## 如何使用（输入与输出）
1. **放置文件**：将待处理的 `.md` 文件及相关素材塞进 **仓库根目录下的 `Clippings/`** 文件夹。
2. **执行处理**：双击运行本目录下的 **`process_images.bat`**。
3. **查看结果**：
   - 原始 `.md` 文件将被修改，其中的图片链接将更新为相对路径。
   - 所有图片将被整理到 **根目录 `Clippings/assets/[文章名]/`** 目录下。
