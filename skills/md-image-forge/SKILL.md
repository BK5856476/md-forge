# MarkdownImageForge 技能说明

本技能用于自动化处理 Markdown 文档中的图像资源，确保文档的自包含性和路径整洁。

## 核心功能
1. **网络图片本地化**：自动提取 MD 文件中的 `http(s)` 链接并下载到本地。
2. **本地路径重构**：将文档引用的本地零散图片迁移至统一的 `assets/文档名/` 目录下。
3. **清理冗余**：自动删除未被引用的孤儿资源文件夹及原始位置的旧图片。
4. **链接修复**：自动更新 MD 源码，将图片链接替换为基于 `assets/` 的相对路径。

## 目录规范
- **根目录下 /Clippings/**: 全局共享输入区，存放待处理的 `.md` 文档和原始本地图片。
- **根目录下 /Clippings/assets/**: 脚本生成的资源存储中心，按文档名分层。
- **scripts/**: 包含核心处理逻辑 `create.py`。

## 如何运行
// turbo
- **自动化运行**：执行 `python scripts/create.py`。
- **手动双击**：运行本目录下的 `process_images.bat`。

## 注意事项
- 处理前建议确认根目录 `Clippings/` 下已放入文件。
- 脚本会自动清理 `Clippings` 目录下已成功迁移的原始图片文件。
