# Markdown 格式规整工具 (md-format-standardizer)

## 简介
这是一个针对 **Markdown 文本排版与格式标准化** 的强力工具，旨在清理采集、剪藏过程中留下的多余痕迹，恢复文档的纯净排版：
- **无噪音处理**：自动清除元数据和分割线。
- **排版优化**：修复代码块标签，列表紧凑化。
- **内容净化**：专业级表情剔除与链接简化。

## 🛠️ 核心功能 (7 大自动化手术)
1. **元数据清理**：识别并彻底删除开头的 Clippings 字典块（ title, source 等）。
2. **分隔线删除**：删除文中任何地方出现的独立成行的 `---` 横线。
3. **文末冗余清理**：切除剪藏附带的翻译声明、发布信息等冗余块，仅在文末保留**作者名**。
4. **链接降噪**：解除 `[文字](URL)` 绑定，只保留可见文本（不处理裸 URL）。
5. **代码块修复**：将孤立在代码块上方的语言标识（如 Python）合并到 ` ```python ` 中。
6. **表情剔除**：基于专业 `emoji` 库，深度清除全篇所有 Unicode 表情符号。
7. **列表紧凑化**：自动删除列表项（- 或 1.）之间的空行，呈现严谨布局。

## 如何运行
1. **放置文件**：确保待规整的 `.md` 文件已放入 **仓库根目录下的 `Clippings/`** 文件夹。
2. **执行处理**：双击运行本目录下的 **`process_format.bat`**。
   - *注：脚本会自动检查并安装必要的 `emoji` 库，无需手动干预。*
3. **查看结果**：脚本将直接修改 `Clippings/` 目录下的源文件。
